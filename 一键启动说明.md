# 博通系统一键启动说明

## 当前启动命令

主程序是一个 `Streamlit` 应用，当前启动命令为：

```bash
streamlit run app.py
```

一键启动脚本会固定使用：

```bash
streamlit run app.py --server.port 8501 --server.headless false --browser.gatherUsageStats false
```

## Windows

1. 将 [启动博通系统.bat](/Users/may/Desktop/stock-analyzer-main/启动博通系统.bat) 复制到桌面，或在桌面为它创建快捷方式。
2. 双击后会：
   - 自动进入项目目录
   - 如检测到 `.venv` 则自动激活
   - 启动 Streamlit
   - 自动打开 `http://localhost:8501`
3. 保持弹出的命令行窗口开启即可持续运行，关闭窗口即停止系统。

## Mac

1. 首次使用先执行：

```bash
chmod +x "/Users/may/Desktop/stock-analyzer-main/启动博通系统.command"
```

2. 将 [启动博通系统.command](/Users/may/Desktop/stock-analyzer-main/启动博通系统.command) 放到桌面后双击。
3. 双击后会：
   - 自动进入项目目录
   - 如检测到 `.venv` 则自动激活
   - 启动 Streamlit
   - 自动打开 `http://localhost:8501`
4. 保持弹出的终端窗口开启即可持续运行，关闭窗口即停止系统。
